# OpenCart Manual Testing Project

## Project Overview

This project demonstrates end-to-end Manual Testing of an OpenCart E-commerce web application.
The goal is to validate core functionalities like user registration, login, cart, checkout, and order management.

---

## Objectives

* To understand real-time software testing process
* To perform functional and UI testing
* To identify and report bugs
* To create complete QA documentation

---

## Modules Covered

* Register Functionality
* Login Functionality
* Logout Functionality
* Forgot Password
* Search Product
* Add to Cart
* Wish List
* Home Page
* My Account
* Change Password
* Order History
* Product Returns

## Test Artifacts

  Document                                            Description                                  
 
Test Plan                              ->     Defines scope, strategy, and testing approach 
Test Scenarios                         ->     High-level test conditions                    
Test Cses                              ->     Detailed step-by-step test cases               
RTM (Requirement Traceability Matrix)  ->     Requirement coverage mapping                  
Test Execution Report                  ->     Pass/Fail results of test cases               
Bug Report                             ->     List of identified defects
